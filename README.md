# Prompt Mods
I've ben wanting an extension like this for a while but never found something that was quite up to my standards so I made Prompt Mods, an AUTOMATIC1111 / Forge / reForge WebUI extension that adds customizable text before and after prompts with support for text stripping *(like when you grab a prompt from PNG Info)* as well as proper integration with Styles, Highres. fix and other extensions like [Dynamic Prompts](https://github.com/adieyal/sd-dynamic-prompts), [ADetailer](https://github.com/Bing-su/adetailer), or [Image Browser](https://github.com/AlUlkesh/stable-diffusion-webui-images-browser).
